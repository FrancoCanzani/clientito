# Inbox and Search

Use this page to work faster in daily focus sessions.

## Inbox model

Duomo splits mail into focused views:

- Focus: active unread inbox work, processed one message at a time.
- To do: messages you intentionally marked for action.
- Screener: new first-contact senders waiting for approval.
- Inbox: active incoming work.
- Done: processed mail.
- Sent: outbound messages.
- Drafts: unsent work.
- Spam/Trash: low-signal or deleted content.

## Focus routine

1. Start in Focus.
2. Act on each thread: reply, archive, snooze, or mark to-do.
3. Use To do for intentional follow-up.
4. Use Inbox and Search when you need the full mailbox surface.

## Search tips

Open search view and query by:

- sender email
- sender name
- subject terms
- body terms

For better results, use 2-4 specific words from subject or sender.

## Filters and subscriptions

- Filters automate labeling/handling.
- Subscriptions view helps clean newsletter noise.

Use both to reduce manual focus work.

## Keyboard and command palette

Press `Cmd/Ctrl + K`:

- `> inbox` jump views
- `> search emails` open search
- `> screener` review new senders
- `> to do` open the action queue
